Facebook API

To use the Facebook API, you first need to get set up with a Facebook Developer account. On your normal homepage, go to the bottom of the navigation bar on the left side. There should be a "DEVELOPER" section. If you already have an account, click "Manage Apps".

Otherwise, put your cursor over the word "DEVELOPER", and click the word "More" that appears to the right. You should see a bar titled "Become a Facebook Developer". Hit the green "Register Now" button, then agree to the terms and click "Register" in the popup.

Once you have an account, create a new app by selecting the "Create a New App" option on the "App" dropdown located on the top menu. In the popup, name your app and click "Create App".

If you want to incorporate a login button, check out the guide here. You can just copy the code from the "Quickstart" and insert it into your own app. You'll also likely need to fix a security issue: On the left side of the app browser on Facebook, select "Settings", then "Advanced", and fill in "Valid OAuth redirect URIs" with "localhost:5000". You should now be able to run your Node app locally and log in to Facebook!

jQuery

jQuery is a Javascript library that simplifies a lot of functions people often want to perform with their websites. You can download version 1.11 from the jQuery website ("Download the compressed, production jQuery 1.11.1").

jQuery makes it easier to change the HTML on your page via Javascript. The dollar sign ($) represents the jQuery library, so you can use it to perform jQuery functions. Here's a sample:

$(document).ready(function(){

$("h2").text("Welcome, friend.");

});

The first line allows you to perform some function after the entire HTML page has loaded. This is the safest way to use jQuery, since it frequently relies on the HTML code (the "document") to perform some function. This code updates the code within all h2 tags it sees in the document with "Welcome, friend."
