How Websites Work

Websites work through a combination of three things: HTML, CSS, and Javascript. Javascript takes care of complicated tasks. We will cover it later. HTML essentially structures the webpage, and CSS styles it.

HTML — HyperText Markup Language

HyperText is text that will hyperlink you to other text.

It is called a Markup Language because HTML is not written as plain text. It is marked up into the form of HTML elements called 'tags.' Tags are denoted by < followed by >.

Brackets

Now we want to install a program called Brackets. Brackets is a code editor for HTML with some really cool features.

One feature we really like about brackets is the 'Live Development Feature.' Live development is enabled by clicking on the lightening bolt on the right side of your Brackets window. It will allow you to see the changes you are making to your HTML code as you make them in your preferred web browser.

index.html

Brackets comes installed with a file called index.html. You should open that file now. This file has the basics for getting you started with HTML in Brackets. We are not going to walk through some of the important tags introduced in this file.

Under the <head> tag you can see tags for <link>. <link>, in general, links the HTML code to an outside source. In this case you can see that it links to an external CSS style sheet.

The <body> element contains all of the contents of the HTML document including the text, hyperlinks, images, etc.

Headers go from <h1> to <h6>. <h1> is the most important heading and <h6> is the least important heading.
