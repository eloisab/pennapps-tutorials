How Websites Work

Websites typically work through a combination of three things: HTML, CSS, and Javascript. Javascript takes care of complicated tasks. We will cover it later. HTML structures the webpage, and CSS styles it.

HTML — HyperText Markup Language

HyperText is text that will hyperlink you to other text.

It is called a Markup Language because HTML is not written as plain text. It is marked up into the form of HTML elements called 'tags.' Tags are denoted by < followed by >.

Brackets

Now we want to install a program called Brackets. Brackets is a code editor for HTML with some really cool features.

One feature we really like about Brackets is the 'Live Development Feature.' Live development is enabled by clicking on the lightening bolt on the right side of your Brackets window. It will allow you to see the changes you are making to your HTML code as you make them as they would appear on the web.

index.html

Brackets comes installed with a file called index.html. You should open that file now. This file has the basics for getting you started with HTML in Brackets. We are now going to walk through some of the important tags introduced in this file. But you should also read through this file as it has many helpful tips for navigating Brackets.

Under the <head> tag you can see tags for <link>. <link>, in general, links the HTML code to an outside source. In this case you can see that it links to an external CSS style sheet.

The <body> element contains all of the contents of the HTML document including the text, hyperlinks, images, etc.

Headers go from <h1> to <h6>. <h1> is the most important heading and <h6> is the least important heading.

Commenting is when you would like to write a note to yourself or someone else that might read your code about what the code is actually doing. It does not affect the output of the code.

You can see from this file that all tags must be opened with <tag> and closed with </tag>.

Other useful tags from this file are:

  • <kbd> — Keyboard Input
  • <samp> — Sample Output from Computer Program
  • <a> — Defines a hyperlink
  • <img> — Import an Image
  • <em> — Emphasized Text
  • <strong> — Important Text
  • <ul> — Bulleted List
  • <li> — Defines List Item
  • <script> — The text written inside this element is not HTML. It is another type of script such as Javascript. It can also point to a different king of script file.
  • <div> — Defines a division in the HTML document
  • <style> — Define appearence of HTML document

CSS — Cascading Style Sheets

CSS desribes the look and formatting of a document written in a markup language (such as HTML). It's super easy to look up how to make certain appearences! If you Google "CSS backgroun-image linear gradient" and go to the W3 Schools result, it will show you the CSS code. W3 Schools is also an excellent reference for HTML tags.

Bootstrap

Bootsrap is a front end developer for webpages. It provides a helpful starting point for building a website. You should download it now. Lots of cool and famous websites were built starting with Bootstrap. You can look at them here. Bootstrap provides code for a lot of website layouts. Go here to see the examples.

You get this code by:
