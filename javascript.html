Javascript

This section covers how to get started with Javascript.

What is a program?

A program is instructions that tell a computer what to do. It is similar to a book in that it is written in a specific language and is made up of words—or code.

Datatypes and Variables

Datatypes are how data is classified. It determines what can be done with data. A variable is a value associated with a symbolic name.
ex. var ten = 10;
The variable in this case is 10, the symbolic name is ten, and the datatype is integer.

Examples of Datatypes:

  • Integers — whole numbers
    ex. 2 or 10249502
  • Floating Point — Approximate real numbers
    ex. π or 1.205
  • Strings — Sequence of characters
    ex. "test" or "1234"
  • Booleans — True and False
    Boolean Logic:
    and — true if both a and b are true, otherwise false
    or — true if either a or b is true (or both are true), otherwise false
    not — reverses the boolean
  • Comparisons — Compares different data types, returns booleans
    ex. == (equals) or != (not equals)

Conditionals

Conditionals control the flow of the program. They go through the program as a decision tree.

If

If boolean, in this case whether powerLevel is greater than 9000, is true then the code is executed.

if (powerLevel > 9000){
var text = "It's over 9000!"; }

If Else

Gives code to execute if boolean is false.

if(time < 20){
text = "good job"; }
else{
text = "too slow"; }

While

The code inside the loop executes continuously as long as boolean remains true.
The ++ adds one to the variable i.

var i = 0;
while(i < 10){
i++; } //executes 10 times

For

You feed the loop an initial condition, a boolean operation, and then how the condition will be changed. The loop executes the code while the boolean remains true.

for(i=0;i<5;i++){
code; } //executes code five times

Arrays

An array is a table of values. The location of each value is called an index. The indices are labelled by integers, starting at 0.

An array can be declared like this:
var cars = ["Saab","Volvo","BMW"];
or this:
var cars = new Array("Saab","Volvo","BMW");

Functions

A function is a block of code designed o perform a specific task. Typically, it is given inputs with which it does some operation to determine the output.

function example(n1,n2){
return n1 * n2; }//output of the function

Objects

Objects are data with customizable properties.
var Person = {name:"Brynn",age:21};
Person.name returns "Brynn" and Person.age returns 21.
